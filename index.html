Currency Exchange Visualization
  1. Select a Country:
    • Use the dropdown menu labeled "Country" to select a country for which you want to view historical currency exchange rates.
  2. View Map:
    • The world map is displayed in the middle of the page, with each country represented by a shaded area. The selected country is highlighted in a different color for easy identification.
  3. View Chart:
    • Below the map, there is a line chart labeled "Historical Currency Exchange Rate." This chart displays the historical exchange rates for the selected country's currency against USD over a predefined set of dates.
  4. Interact with the Map:
    • Hover over a country on the map to see a tooltip with the country's name.
    • Click on a country to update the chart with the historical exchange rates for that country.
  5. Chart Details:
    • The chart provides information on the exchange rate trends over the last twelve months. The x-axis represents the dates, and the y-axis represents the exchange rates against USD.
